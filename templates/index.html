📉 Engagement Quality
📉 Bounce Rates ★
The percentage of single-page sessions. A high bounce rate often indicates that site entrance pages aren't relevant to your visitors.
🚪 Exit Rates ★★
For all pageviews to the page, the percentage that were the last in the session. High exit rates can signal problem areas on your site.
💰 Page Values ★★★
The average value of a page that a user visited before a purchase. This is a key indicator of a page's contribution to revenue.
🎉 Special Day
A value indicating the closeness of the session time to a special day (e.g., Mother's Day). 0 is far, 1 is on the day.
📅 Session Context
📅 Month ★
The month in which the user session occurred. Certain months, like November, show higher purchase rates due to holidays.
👤 Visitor Type
Whether the visitor is new to the site or has visited before. New visitors often have a higher conversion rate.
📆 Weekend
Indicates whether the session took place on a weekend. Weekend shoppers sometimes behave differently than weekday shoppers.
🗺️ Region
The geographic region where the user's session originated. Different regions can have different purchasing habits.
📡 Traffic Type (1–20)
The source of traffic that brought the user to the site (e.g., 1: Direct, 2: Search Engine, 3: Referral).
💻 Technical Info
💻 Operating System
The operating system of the user's device. This can provide insights into the user's technical context.
🌐 Browser
The web browser used by the visitor. Browser type can sometimes correlate with user behavior and conversion.
